## 3. 수평선  (Horizontal Rule) `<hr/>`
아래 줄은 모두 수평선을 만든다. 마크다운 문서를 미리보기로 출력할 때 페이지 나누기 용도로 많이 사용합니다.

```
---
-----------------------
- - - -
***
**********************
* * * *
```

## 8. 코드
* 인라인 코드(Inline Code) : 억음부호(백틱: `)를 이용하는 방벙

억음부호를 이용하여 내용을 강조합니다.   
`억음부호를 이용하여 내용을 강조합니다.`

* `<pre><code>{code}</code></pre>` 이용방식
```

  int main()
  {
    printf("Hello World"/n);

    return 0;
  }
  </code>
</pre>
```

<pre>

  {

    return 0;
  }
  </code>
</pre>

* 블럭 코드(block code,"```")을 이용하는 방법

```
  int main()
  {
    printf("Hello World"/n);
    
    return 0;
  }
```
코드블럭코드("```") 시작점에 사용하는 언어를 선언하여 문법강조(Syntax highlighting)이 가능합니다.

## 9. 링크

* 링크 주소(자동 연결)   
일반적인 URL 혹은 이메일주소인 경우 적절한 형식으로 링크를 형성한다.

외부 링크 : <https://www.example.com/>   
이메일 링크 : <address@example.com>

* 외부 링크

`[링크 텍스트](링크 주소)`   
외부 링크 : [테스트주소](https://www.example.com/)

* 참조 링크

` 링크 텍스트 링크 주소, "부가 설명" `   
참조 링크 : [구글](http://google.com, "검색 사이트")
